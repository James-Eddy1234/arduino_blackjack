The goal of this project was to make a blackjack game. Instead of cards, the game uses dice. This stemmed from the original project idea, which was just rolling a dice with the goal of the rolls summing up to a randomly chosen goal amount. The switch to blackjack instead came from a recommendation comment in Netrun, as this would result in a more fair game because the original idea had an unlikely winning chance. The hardware that was used for this was a breadboard, two buttons, an OLED display, a 4 digit 7 segment display, the Arduino from the lab kit, and some wires and resistors.
